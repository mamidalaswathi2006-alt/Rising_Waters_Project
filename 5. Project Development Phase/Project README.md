# 🌊 Rising Waters – Flood Prediction System

## 📌 Project Overview

Rising Waters is a machine learning-based web application designed to predict the likelihood of floods using weather-related parameters. The system analyzes environmental data and provides instant flood risk predictions through a simple and user-friendly web interface. This project demonstrates the application of machine learning in disaster preparedness and early flood risk assessment.

---

## 🎯 Features

- Predicts flood risk using Machine Learning
- User-friendly web interface
- Instant prediction results
- Analytics page displaying model performance
- Contact page with flood emergency helpline numbers
- Deployed on PythonAnywhere

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/mamidalaswathi2006-alt/Rising_Waters_Project
```

Navigate to the project directory:

```bash
cd Flood_Prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Flask application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

---

## 📊 Machine Learning Model

The flood prediction model is developed using the **Random Forest Classifier** from Scikit-learn. The model is trained on historical weather and flood-related data and saved as `flood_model.pkl` for integration with the Flask application.
